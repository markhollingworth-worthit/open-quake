# Spotify Volume

A community app for Open Quake that controls the Windows volume level of a target process (Spotify by default) using the hardware knob.

## Features

- Adjustable process name (defaults to Spotify)
- Native Windows volume control
- Lightweight C# helper executable

## Installation

Import `spotify-volume.zip` through the Community Apps installer.

## Options

| Option | Description |
|---------|-------------|
| process | Target Windows process name (default: Spotify) |
